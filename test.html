API 요청

파라미터 (JSON 형식)

선택된 오퍼레이션에 따라 필요한 파라미터를 JSON 형식으로 입력하세요.

예시:

  • `areaBasedList`: `{"numOfRows": "10", "pageNo": "1", "areaCode": "1", "contentTypeId": "12", "userLat": "37.566", "userLng": "126.978", "radius": "1000"}`
  • `detailCommon`: `{"contentId": "126588"}` (남산서울타워)
  • `batchDetail`: `{"contentIds": ["126588", "126589", "126590"]}`
  • `locationBasedList`: `{"mapX": "126.978", "mapY": "37.566", "radius": "1000", "contentTypeId": "12"}`
  • `searchKeyword`: `{"keyword": "남산", "contentTypeId": "12"}`
  • `searchFestival`: `{"eventStartDate": "20240101", "numOfRows": "10"}`
  • `areaCode`: `{"numOfRows": "100"}`
