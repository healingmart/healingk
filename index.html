    여기에 API 응답이 표시됩니다. 👋 시작하려면 왼쪽에서 작업을 선택하고 'API 요청 전송' 버튼을 클릭하세요! 📚 사용 가능한 주요 파라미터: • numOfRows: 한 페이지 결과 수 (기본값: 10) • pageNo: 페이지 번호 (기본값: 1) • arrange: 정렬 구분 (A: 제목순, B: 조회순, C: 수정일순, D: 생성일순, E: 거리순) • contentTypeId: 콘텐츠 타입 (12: 관광지, 14: 문화시설, 15: 축제, 32: 숙박, 39: 음식점) • areaCode: 지역 코드 (1: 서울, 6: 부산, 39: 제주도 등) • keyword: 검색 키워드 • contentId: 콘텐츠 ID (상세 정보 조회 시 필요)
